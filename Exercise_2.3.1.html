Model solution

We first show two facts analogous to Lemmas 2.2.2 and 2.2.3:
  1. For any natural number $latex n$, $latex n\times 0 = 0$: We induct on $latex n$. The base case is $latex 0\times 0 = 0$, which follows from the definition of multiplication. Now suppose inductively that $latex n\times 0 = 0$. We want to show $latex (n{{+}\!{+}})\times 0 = 0$. By definition of multiplication and the inductive hypothesis, $latex (n{{+}\!{+}})\times 0 = (n\times 0) + 0 = 0$. This closes the induction.
  2. For any natural numbers $latex n$ and $latex m$, $latex n\times (m{{+}\!{+}}) = (n\times m) + n$: We induct on $latex n$ (keeping $latex m$ fixed). For the case $latex n=0$, we have to show $latex 0\times (m{{+}\!{+}}) = (0\times m) + 0$. But $latex 0\times (m{{+}\!{+}}) = 0$ by definition of multiplication, and $latex (0\times m) + 0 = 0+0=0$, so they are equal. Now suppose inductively that $latex n\times (m{{+}\!{+}}) = (n\times m) + n$. We want to show that $latex (n{{+}\!{+}})\times (m{{+}\!{+}}) = ((n{{+}\!{+}})\times m) + (n{{+}\!{+}})$. By definition of multiplication, $latex (n{{+}\!{+}})\times (m{{+}\!{+}})$ is equal to $latex n\times (m{{+}\!{+}}) + (m{{+}\!{+}})$, which is equal to $latex n\times m + n + (m{{+}\!{+}})$ by the inductive hypothesis. Similarly, we have $latex ((n{{+}\!{+}})\times m) + (n{{+}\!{+}}) = (n\times m + m) + (n{{+}\!{+}})$ by the definition of multiplication. Thus both sides are equal to $latex n\times m + m + n + 1$.
Now we can prove the actual lemma. We fix $latex m$ and induct on $latex n$. For the base case, we need $latex 0\times m = m\times 0$. By the definition of multiplication, the left side is zero, and by fact (1) above the right side is also zero. Now suppose inductively that we have $latex n\times m = m\times n$. We want to show $latex (n{{+}\!{+}})\times m = m\times (n{{+}\!{+}})$. By definition of multiplication, the left side is $latex (n{{+}\!{+}})\times m = n\times m + m$. By fact (2) above, the right side is $latex m\times (n{{+}\!{+}}) = m\times n + m$. Thus we have to show $latex n\times m + m = m\times n + m$, but this follows from the inductive hypothesis.
